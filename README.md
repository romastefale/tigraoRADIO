# tigraoRADIO

Bot Telegram que mostra a música atual do Spotify.

## Comandos
- /start
- /help
- /login
- /logout
- /play

## Gatilhos
- tocando
- tigraofm
- radinho
- qap

## Execução
- Python
- variáveis de ambiente
- comando de run

```bash
python app/bootstrap.py
```

## Observações
- requer login Spotify
- depende da API do Spotify
